# AI Personal Finance Assistant (Track A)

## Project Overview
An AI-powered expense tracking and financial advisory assistant built using Streamlit.

## Current Features
- Streamlit Web Interface
- Screenshot Upload System
- Project Structure Setup
- GitHub Integration

## Upcoming Features
- OCR-based expense extraction
- Automatic expense categorization
- Financial advice using LangChain
- Basic budgeting insights

## Tech Stack
- Python 3.12
- Streamlit
- LangChain (planned)
- SQLite (planned)
- Tesseract OCR (planned)
